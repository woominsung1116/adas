# ADAS 프로젝트 전체 정리

## 한 줄 요약

한국 초등학교 교실 시뮬레이션에서 AI 교사 에이전트가 정상/ADHD/혼동 학생들을 관찰하며 메모리를 축적하고, 경험이 쌓일수록 ADHD 판별과 케어 능력이 향상되는 것을 보여주는 오픈엔디드 시뮬레이션.

---

## 1. 프로젝트 배경

### 참고 논문
- **Agent Hospital** (arXiv:2405.02957) — SEAL 프레임워크
- 가상 병원에서 의사 에이전트가 20,000명 환자를 진료하며 성장
- 핵심 주장: "가상 세계에서 학습한 기술이 현실 문제를 해결할 수 있다"
- MedQA(실제 의사시험)에서 92.22% 달성으로 검증

### 우리 프로젝트
- Agent Hospital의 **교육 도메인 적용**
- 병원 → 교실, 의사 → 교사, 환자 → 학생, 질병 → ADHD
- 한국 초등학교 맥락 (한국 데이터만 사용)

### 연구 목적 (에이전트 성능 향상이 목적이 아님)
1. **에이전트가 뭘 배우는가**: 축적된 감별 전략이 실제 임상 가이드라인과 일치하는가?
2. **현실 전이 가능성**: 학습된 전략이 실제 교사에게 유용한가?
3. **교사 교육 도구**: 신임 교사 연수용 시뮬레이터 가능성
4. **난이도 분석**: 어떤 유형의 ADHD가 가장 판별 어려운가?

## 8. 핵심 문제: 성장이 너무 빠르다

### 현재 문제
- 2-3학급에 100% 도달 (Agent Hospital은 20,000명)
- 원인: 패턴 매칭 문제로 풀리고 있음 (POMDP여야 함)

### CCG 트라이모델 분석 (Codex GPT-5.4 + Gemini 2.5 + Claude Opus 4.6)

3개 모델이 독립적으로 동일 결론 도출:

**Top 5 변경 (합의)**

| # | 변경 | 설명 | 예상 효과 |
|---|---|---|---|
| 1 | **혼동 프로파일** | anxiety, ODD, gifted, 수면부족, 학습장애 추가. 15-25% | PPV 30%에서 시작 |
| 2 | **부분 관찰** | 턴당 3명만 상세관찰 + 행동 가시성 점수 | 정보 획득 4배 느려짐 |
| 3 | **지연 피드백** | 판별 확인 30%만, 나머지 미확인 | 잘못된 원칙 수정 느려짐 |
| 4 | **메모리 노이즈** | 검색 miss 20%, 원칙 승격 7회+ 필요 | 과거 경험 맹신 불가 |
| 5 | **교실 아키타입** | quiet/chaotic/시험/고SES/저SES/학년별 | 전이학습 제한 |

**Gemini 독자 제안 (추가 채택)**
- 가설-검증 루프: 즉시 판별 대신 `의심 → 개입 테스트 → 반응 분석 → 감별`
- 시간적 지속성: 5-10학급에 걸쳐 반복 패턴이어야 판별 허용
- 행동 가시성 점수: 자리이탈=1.0, 공상=0.2

**예상 성장 곡선**
```
Classes  1-10:  severe만 잡음. 민감도 ~40%, PPV ~30%
Classes 10-30:  ADHD vs anxiety/ODD 구분 시작. 민감도 ~60%, PPV ~50%
Classes 30-60:  보정, 검색 품질 개선. 민감도 ~80%, PPV ~70%
Classes 60-100: mild/공존질환에서 미세 향상. 민감도 ~90%, PPV ~85%
```

## 9. 학생 유형 확장 계획

### 현재: 정상 vs ADHD (이진 분류)
### 목표: 7가지 유형 + 학생 간 상호작용

| 유형 | 비율 | 특징 | ADHD와의 혼동 |
|---|---|---|---|
| 정상 (조용한) | ~30% | 안정적, 순응적 | 없음 |
| 정상 (활발한) | ~20% | 에너지 많지만 제어 가능 | 과잉행동형과 혼동 |
| **ADHD** | 6-11% | 3아형 x 3심각도 | (판별 대상) |
| 불안장애 | 5-8% | 부주의+회피+위축 | 부주의형과 혼동 |
| ODD | 3-5% | 반항+공격+불이행 | 과잉행동형과 혼동 |
| 영재 | 3-5% | 지루→산만, 빠른 완료 | 부주의형과 혼동 |
| 수면부족/스트레스 | 5-10% | 일시적 부주의+짜증 | 날마다 변동 |

### 학생 간 상호작용

| 상호작용 | 예시 | 판별 혼동 |
|---|---|---|
| 충돌/싸움 | A가 B를 밀침 → 둘 다 공격적 | 누가 ADHD? |
| 또래 전염 | ADHD가 떠들면 옆 정상도 따라함 | 진짜 ADHD vs 전염 |
| 괴롭힘 | ODD가 불안아동 놀림 → 위축 | 불안 vs ADHD |
| 우정 그룹 | 친한 3명이 수다 → 전부 off-task | 사회적 vs ADHD |
| 좌석 인접 | 산만한 학생 옆 정상도 산만 | 환경 vs 개인 |

### 학생 에이전트 구현 방식
- **인지 아키텍처 에이전트**: Generative Agents 인지 루프(perceive→retrieve→reflect→plan→act)를 사용하되, 각 단계를 파라미터 기반으로 실행(실험용) 또는 LLM으로 실행(시연용) 가능.
- LLM은 교사 에이전트에 필수 사용; 학생 에이전트는 동일 인지 아키텍처를 규칙 기반 또는 LLM 백엔드로 실행 가능 (설정으로 전환)
- Agent Hospital과의 차이: Agent Hospital은 의사·간호사·환자 **전원이 LLM 기반 자율 에이전트** (arXiv:2405.02957). 우리는 비용·속도를 위해 학생은 규칙 기반 실행을 기본으로 함.

## 11. 실험 설계

### 비교군

| 조건 | 설명 |
|---|---|
| Random | 무작위 행동 (하한선) |
| Rule-based (no memory) | 고정 규칙, 메모리 없음 |
| Case Base only | 관찰 기록만, 원칙 학습 없음 |
| **Full (CB + EB)** | **Case Base + Experience Base (핵심)** |
| LLM Teacher (Claude) | 상한선 |

### 분석 관점

1. **학습된 전략 분석**: Experience Base에 축적된 원칙 → 한국 임상 가이드라인과 비교
2. **난이도 층화**: 아형/심각도/혼동 유형별 성능
3. **교실 아키타입별 전이**: 조용한 교실 경험이 혼란스러운 교실에 전이되는가
4. **개입 효율**: oracle vs 에이전트 케어 비용 비교

## 15. 핵심 인사이트

1. **에이전트 성능 수렴 자체가 목적이 아니다** — 수렴 과정에서 뭘 배웠고 현실에 유용한가가 진짜 연구 기여
2. **성장을 느리게 만드는 것이 핵심** — 혼동 프로파일 + 부분 관찰 + 지연 피드백
3. **한국 데이터만 사용** — 학습 환경이 나라마다 다르므로 출처 없는 파라미터 만들지 않음
4. **학생도 인지 아키텍처 에이전트** — 규칙 기반 또는 LLM 백엔드로 실행 가능. Agent Hospital은 환자·간호사·의사 전원이 LLM 에이전트이나, 우리는 비용·속도를 위해 학생은 기본적으로 규칙 기반 실행.
5. **Accuracy 사용 금지** — 유병률 6-11%에서 misleading. AUPRC, PPV, Macro-F1 사용

## 16. 초/중/고 확장 계획

ADHD는 연령에 따라 증상이 변화하므로, 초등뿐 아니라 중·고등까지 확장 가능.

| | 초등 (7-12세) | 중등 (13-15세) | 고등 (16-18세) |
|---|---|---|---|
| **유병률** | 8.5% | 감소 추세 | 더 감소, 사라지지 않음 |
| **주요 증상** | 과잉행동 우세 | 과잉행동↓, 부주의 유지 | 부주의+실행기능 문제 |
| **교실 환경** | 담임 1명, 같은 교실 | 교과 교사 여러 명 | 수능 중심, 자율학습 |
| **혼동 요인** | ODD, 불안 | + 우울증, 품행장애 | + 인터넷중독, 입시스트레스 |
| **관찰 예산** | 3명/턴 | 2명/턴 (교과 시간 짧음) | 1명/턴 (거의 관찰 불가) |
| **한국 데이터** | 풍부 | 중간 | 제한적 |

### 실험 설계
- **횡단**: 각 학교급에서 독립적으로 100학급 → 성장 곡선 비교
- **종단/전이**: 초등 학습 → 중등 전이 → 고등 전이 (이전 경험이 도움 되는가?)
- **Ablation**: Memory vs No-Memory × 학교급

### 구현 방식
- `SCHOOL_LEVELS` 설정만 바꿔서 같은 엔진 재사용
- Phase 1에서 초등 완성 → Phase 2에서 환경 파라미터만 변경해 확장

## 18. 감정 고도화 (Agent Hospital과의 핵심 차별점)

### Agent Hospital vs 우리: 감정의 중요도

| | Agent Hospital (진단) | 우리 (ADHD 교실) |
|---|---|---|
| 핵심 작업 | 증상→검사→진단 (객관적) | 관찰→공감→관계→판별 (감정적) |
| 환자/학생 감정 | 부차적 | **핵심** (감정이 행동을 결정) |
| 의사/교사 감정 | 거의 무관 | **판단에 영향** (소진, 편견) |
| 오판의 결과 | 재진료 | **낙인, 자존감 손상, 트라우마** |
| 개입 방식 | 약 처방 (기계적) | 공감, 관계 형성 (감정적) |

### 학생 감정 모델 (다차원)

현재 `distress_level` 하나 → 8차원 감정 상태로 확장:

```
frustration (좌절감), shame (수치심), anxiety (불안),
anger (분노), loneliness (외로움), excitement (흥분),
trust_in_teacher (교사 신뢰), self_esteem (자존감)
```

### 감정 파라미터 출처 (모두 근거 기반)

| 출처 | 추출 데이터 | 적용 |
|---|---|---|
| K-CBCL 군집분석 4유형 (KCI ART002650863) | 유형별 감정 프로파일 | 초기값 설정 |
| 44편 한국 사례연구 (사전/사후 점수) | 개입별 감정 변화량 | 전이 규칙 |
| 자존감 메타분석 127편 (Springer 2025) | ADHD 자존감 기저값 | self_esteem 초기값 |
| 또래관계 연구 (KCI ART001963695) | 50-60% 또래 어려움 | loneliness 파라미터 |
| O'Leary 1970 | public/private 지적 차이 | shame 효과 차이 |

### 학생 유형별 감정 초기값

```
정상(조용): frustration=0.15, shame=0.10, anxiety=0.15, anger=0.10,
           loneliness=0.10, excitement=0.20, trust=0.60, esteem=0.65

ADHD(과잉): frustration=0.45, shame=0.30, anxiety=0.25, anger=0.40,
           loneliness=0.35, excitement=0.60, trust=0.40, esteem=0.40

불안장애:   frustration=0.25, shame=0.45, anxiety=0.70, anger=0.10,
           loneliness=0.45, excitement=0.10, trust=0.35, esteem=0.30

ODD:       frustration=0.55, shame=0.20, anxiety=0.15, anger=0.65,
           loneliness=0.25, excitement=0.30, trust=0.20, esteem=0.45
```

### 감정이 행동을 결정

같은 "자리이탈"이라도 감정 원인이 다름:
- frustration 높음 → 과제 포기 이탈 (ADHD)
- anxiety 높음 → 화장실 도피 (불안장애)
- anger 높음 → 반항 이탈 (ODD)
- excitement 높음 → 신나서 뛰어다님 (활발한 아이)

교사는 **행동만 보고 감정 원인을 추론**해야 함 → 감정 읽기 능력 = 판별 핵심.

### 교사 감정 모델

```
patience (인내심): 소진되면 강압적 개입 증가
empathy_capacity (공감 여력): 높을수록 감정 정확히 읽음
frustration (교사 좌절): 반복 문제행동에 소진
bias (편향): "활발한 남아 = ADHD" 같은 선입견
```

교사 번아웃 모델: 어려운 학급 후 인내심 불완전 회복 → 다음 학급 판별 정확도 하락.

### 한국 문화적 감정 요소

| 요소 | 설명 | 시뮬레이션 적용 |
|---|---|---|
| **눈치** | 분위기 파악 | 교사 감정 읽기 정확도 |
| **체면** | 남 앞 자존심 | 공개 지적 시 shame 2배 |
| **참기** | 감정 억압 | 안 보이다가 폭발 |
| **정** | 관계적 유대 | trust 높으면 개입 효과 급증 |

### 개입의 감정적 결과 (같은 개입도 감정에 따라 다름)

| 개입 | shame 높은 학생 | trust 높은 학생 | anger 높은 학생 |
|---|---|---|---|
| 공개 칭찬 | 부담 → 위축 | 자존감↑ | 무관심 |
| 공개 지적 | **자존감 추락** | 수용 가능 | 반항 폭발 |
| 1:1 상담 | 안전감 → 열림 | 신뢰 강화 | 점진적 수용 |
| 공감적 인정 | **가장 효과적** | 관계 강화 | 분노 완화 |

### 감정 전염 (Emotional Contagion)

```
A(ADHD) 분노 폭발
  → 옆자리 B(불안) anxiety↑ → 위축
  → C(정상) excitement↑ → 구경하며 소란
  → 교실 전체 불안정
  → 교사 patience↓ → 다음 판단 덜 정확
```

### 감정 관련 추가 메트릭

| 메트릭 | 설명 |
|---|---|
| **Emotional Safety Score** | 오판으로 인한 자존감 손상 누적량 |
| **Empathy Accuracy** | 교사가 학생 감정을 정확히 읽은 비율 |
| **Trust Building Speed** | trust ≥ 0.7 도달까지 턴 수 |
| **Burnout Rate** | 교사 patience ≤ 0.3 빈도 |
| **False Positive Harm** | 오판 후 자존감 변화 |

### 논문 기여

> "Agent Hospital 프레임워크를 감정적으로 민감한 도메인에 적용할 때,
> 진단 정확도뿐 아니라 감정적 영향(emotional impact)까지 평가해야 한다"

이것은 Agent Hospital이 다루지 않은 영역이며 우리 프로젝트만의 독창적 기여.

## 19. Generative Agents 인지 아키텍처 적용

### Stanford Generative Agents (Smallville) 분석 결과

25명 마을 주민 **전원이 독립된 LLM 에이전트**. 각 에이전트가 자기만의 메모리 스트림을 갖고, 매 턴마다 GPT를 3-5회 호출. 2일 시뮬레이션 비용 수천 달러.

### 핵심 인지 루프

```
perceive (지각) → retrieve (회상) → reflect (반성) → plan (계획) → act (행동)
```

### 메모리 검색 공식 (그대로 채택)

```
Score = 0.5 × Recency + 3 × Relevance + 2 × Importance

Recency:    0.99^(현재턴 - 기억된턴)
Relevance:  cosine_similarity(현재상황 embedding, 기억 embedding)
Importance: poignancy 점수 (1-10, 감정적 중요도)
```

### ADHD를 인지 파라미터 차이로 모델링

**같은 인지 아키텍처, 다른 파라미터 = ADHD 모델링**

| 인지 파라미터 | 정상 학생 | ADHD 학생 | 불안 학생 | ODD 학생 |
|---|---|---|---|---|
| `att_bandwidth` (주의 폭) | 3 | 1-2 | 2 | 2 |
| `retention` (기억 유지) | 5 | 2-3 | 4 | 4 |
| `recency_decay` (기억 소멸) | 0.99 | 0.95 | 0.98 | 0.97 |
| `importance_trigger` (반성 임계) | 150 | 50 | 50 | 80 |
| `plan_consistency` (계획 이행) | 높음 | 낮음 | 중간 | 낮음 |
| `impulse_override` (충동 확률) | 0.05 | 0.40 | 0.10 | 0.35 |
| `vision_r` (자극 감지 범위) | 4 | 6 | 3 | 4 |

이 접근의 장점:
- 행동을 if/else로 하드코딩하지 않음 → **파라미터에서 자연스럽게 창발**
- 임상적으로 정확 — ADHD는 별도 시스템이 아닌 같은 인지 시스템의 파라미터 차이
- 연구 주장: "ADHD는 인지 파라미터 차이로 모델링 가능하다"

### 적용 구조

| | 교사 에이전트 | 학생 에이전트 (20명) |
|---|---|---|
| **인지 아키텍처** | 적용 | 적용 (같은 구조) |
| **메모리 축적** | O (학급 간 유지 → 성장) | X (학급마다 리셋) |
| **LLM 사용** | 필수 (의사결정/리포트) | 선택 (규칙 기반 or LLM) |
| **목적** | **성장을 보여주는 것** | **현실적 환경을 만드는 것** |

학생에게 인지 아키텍처를 적용하는 이유는 성장시키려는 게 아니라, 현실적인 교실 환경을 만들어서 교사의 성장이 더 유의미해지도록 하기 위함.

## 22. 시간 단위 설계: 1턴 = 1교시

### 문제: 한 학급이 너무 빨리 끝남

현재 30턴에 학급 종료 → 현실에서 한 학급은 **1년(2학기)**. 교사가 학생을 파악하는 데 1-2개월이 걸리는데, 5턴에 판별이 시작되는 건 비현실적.

### 시간 매핑

```
1턴 = 1교시 (40-45분)
1일 = 5-6턴
1주 = 25-30턴
1학기 = ~500턴 (100일 × 5교시)
1년 = ~1000턴
```

### 현실적 학급 타임라인

```
Turn 1-30 (1주차):
  → 다들 조용함 (새 환경 긴장, 관찰자 효과)
  → 교사가 이름 익히는 중
  → 교사 에이전트: observe 위주, 메모리 거의 비어있음

Turn 30-100 (1개월):
  → ADHD 학생 증상 서서히 표출 (과제 시작 지연, 자리 이탈)
  → 또래 관계 형성 시작
  → 교사 에이전트: 의심 목록 생성 시작

Turn 100-250 (1-2개월):
  → 또래 갈등 발생, 행동 전염
  → ADHD 패턴 뚜렷해짐 (반복 관찰 축적)
  → 혼동 프로파일(불안/ODD)과 구별 필요
  → 교사 에이전트: 판별 시도 시작 (한국 연구 기준 11-20회기, KCI ART002999339)

Turn 250-500 (1학기):
  → 정식 판별 + DSM-5 리포트
  → 케어/개입 시작
  → 교사-학생 trust 형성

Turn 500-1000 (2학기):
  → 케어 효과 관찰 (행동 개선 추이)
  → 관리 완료 여부 판단
  → 유지 효과 검증 (Think Aloud 연구: 4주 유지, KCI ART001124196)
```

### 학급 종료 조건

기존: "전원 관리 완료 시 종료" (너무 빠름)
변경: **1년(1000턴) 경과 시 종료**. 그 안에 얼마나 정확히 판별하고 효과적으로 케어했는가가 성적.

```
학급 성적표:
  - 정확히 판별한 ADHD 학생 수 / 실제 ADHD 수
  - 오판(정상→ADHD) 수 → 자존감 손상 발생
  - 판별까지 걸린 턴 수
  - 케어 후 행동 개선도
  - 학생 자존감 최종값
```

### 속도 계산 (1000턴 기준)

| 백엔드 | 학급당 시간 | 100학급 | 비고 |
|---|---|---|---|
| 규칙 기반 전원 | ~10분 | ~17시간 | 실험용 최적 |
| 로컬 LLM (Qwen 27B, 5090) | ~3시간 | ~300시간 (12.5일) | 자연스러움, 시간 소요 |
| Codex CLI 전원 | ~83시간 | ~8,300시간 | 시연/검증용 |

**로컬 모델(5090 + Qwen 27B)로 전환하면**, 12.5일로 100학급 실험이 가능. 서버에 올려놓고 돌리면 됨. 비용 0원.

### 950턴의 근거

| 기준 | 값 | 근거 |
|---|---|---|
| 한국 초등 수업일수 | 190일 | 초·중등교육법 시행령 |
| 하루 교시 수 | 5교시 (1-2학년) ~ 6교시 (3-6학년) | 교육과정 |
| **1학기** | **~475턴** (95일 × 5교시) | 계산 |
| **1년** | **~950턴** (190일 × 5교시) | 계산 |
| 교사 판별 소요 시점 | ~300턴 (2-3개월 × 5교시) | KCI ART002999339 (11-20회기, 주1-2회 × 2-3개월) |

※ 이전 문서에서 1000턴으로 기재한 것은 반올림한 편의상 수치. 정확히는 **950턴(1년)**.

## 23. 아이디어 간 상충점 분석 및 해결

### 상충 1: 학급 종료 조건

| 섹션 6 (현재 흐름) | 섹션 22 (시간 설계) |
|---|---|
| "전원 관리 완료 → 새 학급" | "1년(950턴) 경과 시 종료" |

**충돌.** → **해결:** 섹션 22가 우선. 950턴 경과 시 종료. "전원 관리 완료"는 성적 지표이지 종료 조건이 아님.

### 상충 2: 학생 에이전트 구현 방식

| 섹션 9 | 섹션 19 | 섹션 20 |
|---|---|---|
| "경량 규칙 기반" | "GA 인지 아키텍처" | "Codex CLI 전원도 가능" |

**혼재.** → **해결:** **인지 아키텍처가 기본 구조**. 각 단계(perceive/plan/act)를 규칙으로 돌릴지 LLM으로 돌릴지는 설정. "규칙 기반"은 인지 아키텍처의 규칙 기반 구현을 의미.

### 상충 3: 부분 관찰 (3명) + 1턴=1교시 (45분)

| 섹션 8 (CCG 권고) | 현실 |
|---|---|
| "턴당 3명만 상세관찰" | 45분이면 더 많이 볼 수 있지 않나? |

**긴장.** → **해결:** "상세 관찰 3명" = 감정/내면까지 파악. 나머지 17명은 **겉보기 행동만** 항상 보임 (자리이탈은 보이지만 공상은 안 보임). 행동 가시성 점수: 자리이탈=1.0(다 보임), 공상=0.2(상세관찰 시에만).

### 상충 4: 감정 모델 (8차원) vs 인지 파라미터

| 섹션 18 | 섹션 19 |
|---|---|
| 감정: frustration, shame, anxiety... | 인지: att_bandwidth, impulse... |

**관계 불명확.** → **해결:** 인지 파라미터 = **고정 특성** (ADHD면 항상 att_bandwidth 낮음). 감정 = **동적 상태** (같은 ADHD도 기분에 따라 다름). 감정이 인지 파라미터를 일시적으로 조절:
```
실제 att_bandwidth = 기본값(2) - frustration(0.5) × 0.5 = 1.75
→ 좌절감 높을수록 주의력 더 떨어짐
```

### 상충 5: "5-10학급 지속성" vs 학급마다 학생 리셋

| 섹션 8 (Gemini) | 섹션 22 |
|---|---|
| "5-10학급 걸쳐 반복이어야 판별" | 학급 바뀌면 학생 리셋 |

**충돌.** → **해결:** "같은 학생"이 아니라 "같은 패턴". Case Base 검색으로 "이전 학급에서 비슷한 행동 패턴을 본 적 있다" → 자연스럽게 해결. 에이전트는 패턴을 학습하는 것이지 학생 개인을 기억하는 게 아님.

### 상충 6: "한국 데이터만" vs 감정 세부 수치

| 원칙 | 섹션 18 |
|---|---|
| "출처 없는 파라미터 만들지 않음" | "shame -0.10" 같은 세부 수치 |

**긴장.** → **해결:** 논문에서 **방향과 대략적 크기**만 추출. 정확한 수치는 calibration parameter로 명시. Limitations에 "일부 파라미터는 시뮬레이션 내 보정을 통해 설정" 기술.

### 상충 7: Phase 구분 (5턴/15턴) vs 950턴 타임라인

| 섹션 6 (현재 코드) | 섹션 22 |
|---|---|
| "Phase 1: 1-5, Phase 2: 6-15, Phase 3: 16+" | "Turn 300에서 판별 시작" |

**충돌.** → **해결:** Phase 기준을 950턴에 맞춰 재설계:
```
Phase 1 (관찰):     Turn 1-100   (~1개월, 이름 익히기+전체 훑기)
Phase 2 (스크리닝): Turn 100-300 (~2-3개월, 의심 학생 집중 관찰)
Phase 3 (판별):     Turn 300-475 (~학기 중후반, DSM-5 리포트)
Phase 4 (케어):     Turn 475-700 (~2학기 초중반, 개입+행동 개선)
Phase 5 (유지/재발): Turn 700-950 (~2학기 후반, 유지 확인+재발 감지+재개입)
```

**Phase 5가 중요한 이유 — 재발 가능성:**
- 토큰강화 ABAB 설계: 중재 철회 시 문제행동 다시 증가 (학위논문)
- 자기점검: 유지 단계에서 부분 증가 가능 (KCI ART002794420)
- 교사가 다른 학생에 집중하면 기존 케어 대상의 관심이 줄어 재발
- **재발 감지 + 재개입 능력 = 교사 에이전트 성장의 핵심 지표**

```
예상 타임라인:
Turn 300:  판별 완료
Turn 350:  개입 시작 → 행동 개선
Turn 500:  개입 효과 안정화
Turn 600:  교사가 다른 학생에 집중 → 관심 줄어듦
Turn 700:  ★ 재발 (compliance 하락)
Turn 750:  교사가 재발 감지 → 재개입
Turn 900:  유지 확인
Turn 950:  학급 종료, 최종 평가
```

### 상충 없이 호환되는 아이디어들

| 아이디어 A | 아이디어 B | 관계 |
|---|---|---|

### 핵심 테이크어웨이

1. **"컨셉 > 증거"** — 현재 주장 범위가 증거 수준을 초과함. 주장을 좁혀 실제로 검증 가능한 범위로 한정해야 함.
2. **Expert Audit + Ablation이 필수** — Rigor 점수가 낮은 원인. Case Base / Experience Base 각각의 기여도를 ablation으로 분리 측정해야 논문 수준 주장 가능.
3. **감정 안전성 각도가 가장 독창적** — 오판이 아동 자존감에 미치는 영향을 평가하는 것은 Agent Hospital / Generative Agents가 다루지 않은 영역. 이 부분이 가장 높은 Novelty를 받은 이유.
4. **Agent Hospital 대비 좁지만 깊은 범위** — Agent Hospital은 의사 성장(진단 정확도)에 집중. 우리는 ADHD 판별 + 감정 안전성까지 다루어 범위는 좁지만 도메인 깊이가 더 높음.

### 순환 논리(Circular Logic) 문제와 해결

**문제:** 학생을 DSM-5 기준으로 만들고, 교사에게 DSM-5 기준으로 찾으라고 시킴. 에이전트가 "학습"한 것이 실제 학습인가, 아니면 출제자가 넣어둔 정답을 찾은 것뿐인가?

```
DSM-5로 학생 생성 → DSM-5로 판별 지시 → DSM-5 패턴 "발견"
→ "학습했다!" (비판: 순환 아닌가?)
```

**완전한 순환이 아닌 이유:**
- 혼동 프로파일(불안/ODD/영재)도 비슷한 행동을 보임 → "어떤 맥락에서 ADHD인지" 감별은 코드에 하드코딩 안 됨
- 부분 관찰 → "누구를 언제 관찰해야 하는지" 전략은 에이전트가 학습해야 함
- 개입 전략 선택 → DSM-5는 진단 기준만 있고 "어떤 개입이 효과적인지"는 안 알려줌

**해결 방법 (구현 필요):**

| 방법 | 설명 | 효과 | 난이도 |
|---|---|---|---|
| **Expert Audit** | Class 1 / Class 100의 Experience Base를 실제 교사 5명에게 이중맹검 평가. "어느 조언이 더 유용한가?" | 순환을 깨는 **외부 검증**. 이것만 하면 논문 수준 급상승 | 중 |
| **Ablation** | 메모리 없음 / Case Base만 / Experience Base만 / 전체 비교 | 학습이 의미 있는지 정량적 증거 | 하 |
| **반-순환 메트릭** | DSM-5 정확도 대신 **"코드에 없는 패턴을 에이전트가 발견했는가"** 분석 | 창발적 학습의 증거 | 중 |
| **실제 사례 검증** | 한국 사례연구 44편의 행동 기술을 시뮬레이션에 넣고 에이전트가 맞추는지 | Agent Hospital의 MedQA에 해당하는 외부 벤치마크 | 상 |
| **감정 Ablation** | 감정 모델 유/무 비교. 정확도 같지만 자존감이 다르면 → "공감이 AI에 중요" 증명 | 독창적 기여 강화 | 하 |

**논문에서의 표현:**
- 주장 범위를 **"시뮬레이터 내 메모리 기반 학습 연구"**로 한정
- "교사 교육 도구" / "현실 전이"는 **"가능성 시사"**로만 기술
- Expert Audit를 실시하면 주장을 확장할 수 있음

# 25. Dimensional 파라미터 모델 & Autoresearch Calibration

## 25.1 배경 — 왜 hand-tuned 초기값이 문제인가

### 현재 구조의 결함
- `COGNITIVE_PRESETS`, `EMOTIONAL_PRESETS` 의 값들은 논문을 참고했다고 주석에 기재되어 있으나, 실제 **수치는 hand-tuned**
- `self.state` 초기값 `{distress_level: 0.2, compliance: 0.7, attention: 0.6, escalation_risk: 0.1}` 은 **모든 프로파일 동일** — ADHD든 normal이든 첫 턴 관찰값이 같음 (명백한 모델링 오류)
- 수치의 출처를 리뷰어가 물으면 방어 불가

### 대안 — RDoC + Computational Psychiatry 접근
정신질환을 **카테고리가 아니라 연속선(dimensional)** 으로 보고,
모든 프로파일을 **normal baseline + delta** 로 표현하는 구조로 리팩토링.

```
normal_quiet (baseline)
├─ adhd_inattentive    = baseline + Δ(att↓, shame↑, ...)
├─ adhd_hyperactive    = baseline + Δ(impulse↑, excite↑, ...)
├─ adhd_combined       = baseline + Δ(I) + Δ(H) + interaction
├─ odd                 = baseline + Δ(anger↑, trust↓, ...)
├─ anxiety             = baseline + Δ(anxiety↑, se↓, ...)
└─ ...
```

## 25.2 이론적 근거

### 25.2.1 RDoC (Research Domain Criteria)
NIMH(미국 국립정신건강원)의 **공식 연구 프레임워크** (2010~).

**핵심 주장**:
> 정신질환은 이분법적 진단(ADHD O/X)이 아니라
> 연속적인 신경생물학적 차원들(주의, 실행기능, 각성 등)의 극단값이다.

**출처**:
- Insel et al. (2010) "Research Domain Criteria (RDoC): toward a new classification framework for research on mental disorders." Am J Psychiatry 167(7):748-751
- Cuthbert & Insel (2013) "Toward the future of psychiatric diagnosis: the seven pillars of RDoC." BMC Medicine 11:126
- NIMH RDoC Matrix (공식 웹사이트)

### 25.2.2 Dimensional Psychopathology 실증 근거

**ADHD가 실제로 dimensional 이라는 통계적 증거** (taxometric analysis):

- **Haslam et al. (2006)** "The latent structure of attention-deficit/hyperactivity disorder: a taxometric analysis." Aust N Z J Psychiatry 40(8):639-647
  → ADHD는 categorical이 아닌 **dimensional** 로 나타남

- **Marcus & Barry (2011)** "Does attention-deficit/hyperactivity disorder have a dimensional latent structure? A taxometric analysis." J Abnorm Psychol 120(2):427-442
  → 메타분석 결과 ADHD는 연속 분포의 극단

- **Larsson et al. (2012)** "The heritability of clinically diagnosed attention deficit hyperactivity disorder across the lifespan." Psychol Med 44(10):2223-2229
  → 쌍둥이 연구: ADHD 유전적 위험은 연속 분포

### 25.2.3 Computational Psychiatry — 파라미터 shift 모델

정신질환을 **정상 인지모델의 파라미터 변화**로 표현하는 학문 분야.

**핵심 논문**:
- **Huys et al. (2016)** "Computational psychiatry as a bridge from neuroscience to clinical applications." Nat Neurosci 19(3):404-413
- **Friston et al. (2014)** "Computational psychiatry: the brain as a phantastic organ." Lancet Psychiatry 1(2):148-158
- **Montague et al. (2012)** "Computational psychiatry." Trends Cogn Sci 16(1):72-80
- **Stephan et al. (2016)** "Computational approaches to psychiatry." Curr Opin Neurobiol 25:85-92

**분야 저널**: *Computational Psychiatry* (MIT Press, 2017~)

### 25.2.4 ADHD 내부 이론 — 차원 모델 일관성

- **Barkley (1997)** "Behavioral inhibition, sustained attention, and executive functions: constructing a unifying theory of ADHD." Psychol Bull 121(1):65-94
  → ADHD = 단일 연속 파라미터(behavioral inhibition)의 dysfunction

- **Sonuga-Barke (2003)** "The dual pathway model of AD/HD: an elaboration of neuro-developmental characteristics." Neurosci Biobehav Rev 27(7):593-604
  → Dual pathway: 실행기능 + 보상 지연 두 연속 차원

### 25.2.5 논문에 쓸 핵심 문장 (복붙 가능)
> "We adopt the Research Domain Criteria (RDoC) framework (Insel et al., 2010), which conceptualizes mental disorders as dimensional variations rather than discrete categories. Consistent with computational psychiatry approaches (Huys et al., 2016; Montague et al., 2012), we model ADHD subtypes, ODD, and anxiety as parameter shifts from a normative cognitive baseline. This dimensional approach is empirically supported by taxometric analyses showing ADHD lies on a continuous distribution rather than representing a categorical boundary (Haslam et al., 2006; Marcus & Barry, 2011)."

## 25.3 Comorbidity (질환 공존) 근거

### 25.3.1 왜 공존을 모델링해야 하는가
실제 임상에서 ADHD는 단독보다 **다른 질환과 공존하는 경우가 훨씬 많음**. 단순 프로파일(ADHD만, 불안만)로는 현실 교실을 재현할 수 없음.

### 25.3.2 역학적 근거
- **Larson et al. (2011)** "Patterns of comorbidity, functioning, and service use for US children with ADHD." Pediatrics 127(3):462-470
  → ADHD 아동의 **약 67%**가 하나 이상의 공존질환 보유

- **MTA Cooperative Group (1999)** "A 14-month randomized clinical trial of treatment strategies for ADHD." Arch Gen Psychiatry 56(12):1073-1086
  → ADHD 아동 중 ODD 공존 40%, 불안 공존 34%, 우울 공존 14%

- **Jensen et al. (2001)** "ADHD comorbidity findings from the MTA study." J Am Acad Child Adolesc Psychiatry 40(2):147-158
  → 공존 패턴은 단일 질환과 **다른 임상 경과** 를 보임 (치료 반응 차이)

### 25.3.3 한국 데이터
- **김붕년 등 (여러 KCI 논문)** → 한국 ADHD 아동의 ODD 공존률 약 35-45%, 불안장애 공존률 25-30% 보고
- Korean Child Mental Health Survey (KCMHS) — 전국 단위 유병률 및 공존 패턴 제공

### 25.3.4 구조적 반영 — 단순 합산이 아닌 이유
- **ADHD + 불안**: 집중력 저하는 ADHD에서 오는데, **회피 행동**은 불안 기여 → 단순히 delta 합산하면 "과잉 집중력 저하"로 오류
- **ADHD + ODD**: 두 질환의 충동성이 시너지 → 단순 합산보다 **강한 충동성** 으로 관찰됨
- **ADHD + 우울**: self_esteem과 task_initiation_delay가 상호작용 → 우울이 ADHD의 동기 문제를 악화

→ **명시적 interaction term 필요**

### 25.3.5 구현 전략
```python
PROFILE_DELTAS = {
    "adhd_inattentive": {...},
    "anxiety": {...},
    # Comorbidity — base deltas 합산 + 추가 interaction
    "adhd_i_plus_anxiety": {
        "base": ["adhd_inattentive", "anxiety"],
        "interaction": {                    # autoresearch가 학습
            "shame": +0.10,                 # 합산보다 더 높음
            "frustration": -0.05,           # 일부 상쇄
            "plan_consistency": -0.05,
        },
    },
    "adhd_i_plus_odd": {...},
    "adhd_h_plus_odd": {...},
    "adhd_plus_anxiety_plus_depression": {...},
}
```

**핵심 트릭**: Autoresearch 루프에 **sparsity regularization** 추가 → 꼭 필요한 interaction만 살아남음.
- 이론 근거: LASSO (Tibshirani 1996), Sparse modeling in comp psychiatry (Stephan et al.)

## 25.4 Autoresearch Calibration 전체 설계

### 25.4.1 역할 분담

| 역할 | 인간이 결정 | Autoresearch가 결정 |
|---|---|---|
| 이론적 구조 (RDoC, delta 구조) | ✅ | ❌ |
| 파라미터 **범위** | ✅ (문헌 근거) | ❌ |
| 최적화 **목표 수치** | ✅ (한국 역학) | ❌ |
| 구체적 **값** | ❌ | ✅ |
| 공식 **형태** 선택 | 후보 제시 | 최적 선택 |
| Interaction term 필요 여부 | 후보 제시 | 자동 판단 (sparsity) |

### 25.4.2 최적화 목표 (Calibration targets)

**Primary targets (한국 역학)**:
- ADHD-inattentive 유병률 ≈ 5.0% (KCI ART001701933)
- ADHD-hyperactive/combined 유병률 ≈ 2.3%
- ODD 공존률 ≈ 35-45%
- 불안 공존률 ≈ 25-30%

**Secondary targets (임상 특성)**:
- Teacher screening sensitivity 상한 ≈ 0.70 (메타분석 ballpark)
- ADHD vs normal 정서조절 effect size (Cohen's d) ≈ 0.85
- K-CBCL behavior frequency ratio

**Held-out targets (검증용, calibration에서 제외)**:
- Strategy effectiveness profile
- 시간대별 (1교시 vs 5교시) 정서 변화 패턴
- 중재 후 compliance 변화량

### 25.4.3 다양한 상황 (Scenario diversity) 활용

**핵심 관찰**: 우리 시뮬레이터는 **950턴 × 다양한 classroom archetype** 을 지원함.
→ **한 학급 내에서 이미 다양한 상황이 발생** (아침 활발, 오후 피로, 체육 후 흥분, 시험 전 긴장 등)
→ 이걸 **calibration target 다양화** 에 활용 가능

**Scenario-diverse calibration targets**:
- 같은 ADHD 학생의 **행동 프로필이 상황별로 어떻게 다른가**
- Chaotic archetype vs Compliant archetype 에서 **ADHD 식별 정확도 차이**
- Stress event 직후 vs 평상시 정서 회복 시간
- Peer interaction 유무에 따른 compliance 변화

**논문 기여**: 단일 scenario가 아닌 **다양한 상황 전반에 걸쳐 calibration** 하므로, 외삽(extrapolation) robustness 가 강화됨.

### 25.4.4 Model Form Discovery (공식 자동 탐색)

**문제**: 감정/행동 업데이트 공식 자체가 틀릴 수 있음 (linear vs multiplicative vs sigmoidal)

**해결**: Autoresearch 루프에 **공식 후보**도 탐색 대상에 포함.

```python
MODEL_FORMS = {
    "emotion_update": {
        "linear":       lambda e, event: e + alpha * event,
        "sigmoidal":    lambda e, event: sigmoid(logit(e) + alpha * event),
        "multiplicative": lambda e, event: e * (1 + alpha * event),
        "barkley":      lambda e, event: barkley_model(e, event),
    },
    "attention_decay": {...},
    "impulse_override": {...},
}
```

**Autoresearch loop**:
1. 각 form 조합에 대해 파라미터 calibration
2. 각 조합의 calibration_loss + held-out loss 비교
3. **Information criterion** (AIC/BIC) 으로 모델 복잡도 대비 적합도 평가
4. 최종 선택: 복잡도 페널티 반영한 best form

**이점**:
- 단일 form 가정이 틀려도 대처 가능
- 논문에 "우리는 여러 공식을 비교해서 데이터에 맞는 걸 선택했다" 라고 쓸 수 있음
- Model form 자체가 **자동 발견된 기여** 가 됨

### 25.4.5 식별 불가능성 (Identifiability) 대응

**Multi-start autoresearch**:
- N번 (예: 30번) 다른 초기값에서 calibration 실행
- 결과 solution 분포 분석
- 단일 수렴 → unique solution
- 여러 cluster → "multiple regimes" 솔직히 보고

**이론적 제약 주입**:
- ADHD-I: |Δattention| > |Δimpulse|
- ADHD-H: |Δimpulse| > |Δattention|
- Anxiety: Δanxiety > 0 (단조성)
- trust_in_teacher: 대부분 프로파일에서 delta ≤ 0

**문헌 prior 주입**:
- K-CBCL raw data 확보 시 Gaussian prior 설정
- Prior mean ± SD 안에서만 탐색

### 25.4.6 외삽 (Extrapolation) 방어

**Cross-scenario validation**:
- Calibration scenario: Checkmate archetype
- Held-out scenario: Stupp archetype, 신규 archetype
- Held-out performance 보고

**다양한 target 수집**:
- 기본 target 5개 → 확장 10-15개
- 많을수록 robust

**Human Expert Audit** (HANDOFF 로드맵 연동):
- 시뮬레이션 궤적 5명 교사 평가
- Likert + qualitative
- 숫자로 못 잡는 오류 질적 검증

## 25.6 한계와 솔직한 표현 (논문에 쓸 것)

**이건 숨기지 말고 Discussion에 명시**:

1. **Linear additivity 가정**: 실제 non-linear 상호작용은 interaction term + autoresearch calibration 으로 부분 보완
2. **Independent dimensions 가정**: 실제 상관된 변화는 calibration 이 system-level 에서 흡수
3. **식별 한계**: 여러 parameter regime 이 동일 출력 가능 → multi-start 로 드러내고 보고
4. **외삽 한계**: Calibration target 밖 상황에서는 성능 하락 가능 → cross-scenario + expert audit 로 방어
5. **문헌 기반 target 의 본질적 불확실성**: 메타분석 수치 자체가 분산 있음 → sensitivity analysis 로 robustness 확인

**논문의 honest framing**:
> "Our calibration approach trades individual parameter interpretability for system-level empirical accuracy. Specific parameter values should not be interpreted as biological ground truth, but rather as internally consistent configurations that reproduce established epidemiological and clinical targets within the modeled framework."

## 25.7 Comorbidity 확장 — 문헌 근거와 구현할 공존 패턴

### 25.7.1 왜 공존을 확장해야 하는가

단순 프로파일(ADHD만, 불안만, ODD만)로는 실제 교실을 재현할 수 없음. 임상 문헌 대부분에서 ADHD 아동의 **50-75%가 하나 이상의 공존질환** 을 보고함. 공존은 예외가 아니라 **기본값** 에 가까움.

### 25.7.2 주요 공존 조합과 실증 근거

#### (1) ADHD + ODD — 가장 흔함
**역학**:
- **MTA Cooperative Group (1999)**: ADHD 아동의 **40%** 가 ODD 동반
- **Biederman et al. (1991)**: "Comorbidity of attention deficit hyperactivity disorder with conduct, depressive, anxiety, and other disorders." Am J Psychiatry 148(5):564-577
- **Angold et al. (1999)**: "Comorbidity." J Child Psychol Psychiatry 40(1):57-87 — 메타분석; ADHD-ODD OR ≈ 10.7

**임상 특징**:
- 단순 합산 이상의 **충동성+분노 시너지**
- 교사의 지시 저항이 극단적
- 치료 반응이 가장 나쁜 조합

**한국 데이터**:
- 김붕년 등 → 한국 ADHD 아동의 ODD 공존률 약 **35-45%**

#### (2) ADHD + 불안장애
**역학**:
- **MTA (1999)**: ADHD 아동의 **34%** 가 불안장애 동반
- **Jensen et al. (2001)**: "ADHD comorbidity findings from the MTA study" — 불안 공존 ADHD는 단독 ADHD와 다른 **치료 반응 프로파일**
- **Pliszka (1989)**: "Effect of anxiety on cognition, behavior, and stimulant response in ADHD." J Am Acad Child Adolesc Psychiatry 28(6):882-887 — 불안이 ADHD 충동성을 일부 억제

**임상 특징**:
- "Working memory 이중 손상" (ADHD + 불안 모두 WM 감소)
- 회피 행동이 ADHD 충동성을 가려서 **진단 지연** 발생
- Internalizing 증가 → 수업 중 조용히 멍 때리는 패턴

**한국 데이터**:
- 한국 ADHD 아동 불안장애 공존률 약 **25-30%**

#### (3) ADHD + 우울
**역학**:
- **Jensen et al. (2001)** MTA: ADHD의 **14%** 가 우울 동반
- **Angold & Costello (1993)**: "Depressive comorbidity in children and adolescents." Am J Psychiatry 150(12):1779-1791
- **Daviss (2008)**: "A review of co-morbid depression in pediatric ADHD." J Child Adolesc Psychopharmacol 18(6):565-571 — 공존률 6-38%

**임상 특징**:
- Task initiation delay 극단화
- Self-esteem 최저
- 자살사고 위험 증가 (기본 ADHD 대비)

#### (4) ADHD + 학습장애 (LD)
**역학**:
- **DuPaul et al. (2013)**: "Comorbidity of LD and ADHD: implications of DSM-5." J Learn Disabil 46(1):43-51 — 공존률 **25-40%**
- **Larson et al. (2011)** Pediatrics: 학령기 ADHD 아동의 **46%** 가 학습장애 진단

**임상 특징**:
- 부주의 증상과 LD의 학업 실패를 구분하기 어려움
- 교사 식별 정확도가 가장 낮음
- Frustration과 shame이 급격히 축적

#### (5) ADHD-Inattentive + 불안 — 조용한 공존
**왜 중요**:
- 교실에서 **"조용한 ADHD"** 의 대부분이 이 조합
- 외현화 증상이 없어 교사 눈에 띄지 않음
- DSM-5가 놓치는 "소외된 ADHD" 의 전형 (특히 여아)

**문헌**:
- **Quinn & Madhoo (2014)**: "A review of attention-deficit/hyperactivity disorder in women and girls." Prim Care Companion CNS Disord 16(3) — 여아 ADHD의 inattentive + 불안 공존 패턴
- **Ohan & Visser (2009)** — 교사 판단에서 inattentive 여아가 가장 놓침

#### (6) ADHD-Hyperactive + ODD — 파괴적 공존
**왜 중요**:
- 시뮬레이션에서 "가장 시끄러운" 조합
- 교사가 쉽게 식별하지만, 개입 효과가 가장 낮음
- 또래 갈등의 원인 80% 이상

**문헌**:
- **Burke et al. (2002)**: "Oppositional defiant disorder and conduct disorder: a review of the past 10 years." J Am Acad Child Adolesc Psychiatry 41(11):1275-1293
- MTA study 추적 관찰: 이 조합이 장기 예후 최악

#### (7) 불안 + 우울 (공통 내재화)
**역학**:
- **Kessler et al. (2005)** 대규모 역학: 공존률 **40-60%**
- 아동기에서도 공통 factor ("p factor" 논의, Caspi et al. 2014)

**임상 특징**:
- ADHD와 혼동되지 않지만 교실에서 비슷한 "위축" 행동
- ADHD 감별진단에서 중요한 **distractor profile**

### 25.7.3 구현할 공존 프로파일 목록

우선순위 순:
1. `adhd_i_plus_anxiety` — "조용한 ADHD" (이미 추가됨)
2. `adhd_h_plus_odd` — "파괴적 ADHD" (이미 추가됨)
3. `adhd_c_plus_odd` — combined + ODD (추가 필요)
4. `adhd_i_plus_ld` — ADHD-I + 학습장애 유사 프로파일 (추가 필요)
5. `adhd_plus_depression` — 우울 공존 (추가 필요)
6. `anxiety_plus_depression` — 내재화 distractor (추가 필요)

### 25.7.4 구조적 원칙 — Additive base + Interaction term

```python
PROFILE_DELTAS["adhd_i_plus_anxiety"] = {
    "_base": ["adhd_inattentive", "anxiety"],  # 기본은 합산
    "cognitive": {...},  # 추가 interaction (autoresearch가 학습)
    "emotional": {...},
    "observable": {...},
    "_sources": "MTA 1999; Jensen et al. 2001; Quinn & Madhoo 2014",
}
```

**Sparsity regularization**: Interaction term이 0에 가까우면 페널티 면제 → 꼭 필요한 interaction만 남음.

### 25.7.5 논문 기여 방향
> "Unlike prior classroom simulations that model disorders as isolated categorical presets, our framework captures comorbidity as dimensional overlap with explicit interaction terms. Autoresearch-driven sparsity regularization discovered that only specific comorbidity combinations (e.g., ADHD-Inattentive + Anxiety, ADHD-Hyperactive + ODD) required non-additive interaction terms, while others were well-explained by additive composition."

## 25.8 Situational Events — 950턴 내에서 다양한 상황 이벤트 주입

### 25.8.1 왜 필요한가

현재 시뮬레이션은 **정상 수업 루틴** 만 모델링. 실제 교실은:
- 학기 주기에 따른 긴장도 변화 (시험 전, 방학 전)
- 갑작스러운 이벤트 (싸움, 발표, 전학생)
- 계절/날씨 변화
- 가정 stressor 반영

이런 다양성 없이 calibration 하면 **"평범한 수업 시간"에만 맞춰진** 모델이 됨. 외삽 불가능.

### 25.8.2 기회 — 950턴은 충분히 길다

한 학급 950턴 ≈ **한 학기 전체** 를 시뮬레이션할 수 있는 길이.
→ 학기 내 이벤트를 자연스럽게 녹일 수 있음.
→ Cross-scenario validation을 **한 학급 내부에서** 수행 가능.

### 25.8.3 이벤트 카테고리 제안

#### (A) 학기 주기 (Temporal cycle)
| 기간 (turn 구간) | 특성 | 영향 |
|---|---|---|
| 1-150 (학기 초) | 적응기 | 불안 ↑, 신규성 흥분 |
| 150-350 (안정기) | 루틴 정착 | 평상시 baseline |
| 350-500 (중간고사) | 시험 스트레스 | 불안 ↑↑, 주의 변동 |
| 500-650 (중간 후) | 이완 + 피로 | 산만 ↑ |
| 650-800 (기말 준비) | 스트레스 축적 | Escalation ↑ |
| 800-950 (기말 + 방학) | Peak stress → 이완 | 극단 변동 |

근거:
- **Brown (1996)** 학교 기반 stress 연구
- Korean 교육환경 연구 (수능/내신 중심 주기)

#### (B) 이벤트 Trigger (one-time)
- **새 학생 전학** (turn 100, 300, 500): 사회적 reshuffle
- **또래 갈등** (확률 기반): 전 학급에 영향
- **발표/조별 활동** (주기적): 불안 학생에 큰 자극
- **대체 교사 투입** (1-2 epoch): 교사 요인 혼란
- **학교 행사** (운동회, 축제): 흥분 ↑

근거:
- **Lazarus & Folkman (1984)** Stress and coping
- Korean 교실 문화 연구 (발표 공포 문헌)

#### (C) 날씨/계절 (Continuous modulation)
- **더운 날**: 집중 ↓, 피로 ↑
- **비 오는 날**: 우울 증가 (SAD literature)
- **겨울 아침**: 각성 ↓, 반응 지연

근거:
- **Denissen et al. (2008)** Weather and mood
- **Kasper et al. (1989)** Seasonal affective disorder

#### (D) 가정 배경 Stressor (각 학생별 hidden state)
- 일부 학생에게 **home_stress** hidden param
- 특정 이벤트 후 ADHD-like 증상 증가
- 교사가 ADHD와 감별해야 하는 **distractor profile**

### 25.8.4 구현 방법

#### 방법 1: 환경 modulation 벡터
```python
class ClassroomEnvironmentV2:
    def step(self, turn):
        ctx = ClassroomContext()
        # Base context
        ...
        # Apply situational modulation
        mod = self.get_situational_modulation(turn)
        ctx.global_stress += mod["stress"]
        ctx.attention_noise += mod["attention_noise"]
        ctx.social_disruption = mod.get("social_event")
        return ctx
```

#### 방법 2: Event injection schedule (설정 가능)
```yaml
# .harness/situational_schedule.yaml
events:
  - turn: 350
    type: "exam_stress_onset"
    duration: 100
    effect: {stress: +0.3, anxiety_global: +0.2}
  - turn: 100
    type: "new_student"
    effect: {social_reshuffle: true}
  - turn: 200
    type: "peer_conflict"
    participants: [S03, S07]
```

#### 방법 3: Classroom Archetype 내재화
현재 archetype (compliant/chaotic/stressed/etc.)에 **turn-dependent modulation curve** 추가:
```python
ARCHETYPE_SCHEDULES = {
    "exam_focused": {
        "stress_curve": [0.1, 0.3, 0.6, 0.8, 0.5, 0.9, 1.0, 0.3],  # turn에 따라
        "events": ["midterm_week_350", "finals_week_800"],
    },
}
```

### 25.8.5 Autoresearch와 연계

**Target 다양화**:
- 현재: "유병률 5%, 교사 정확도 70%"
- 확장: "평상시 정확도 70%, **시험 주간 정확도 60%**, **학기 초 정확도 55%**"

각 상황별 target을 설정하면 → Calibration이 **상황 인식 가능한** 파라미터를 자동으로 찾음.

This tests whether identification accuracy degrades under expected stressors, providing a more ecologically valid evaluation of AI teacher performance."

## 25.10 Situational Modulation — 각 이벤트의 파라미터 변화 근거

### 25.10.1 핵심 원칙
각 상황은 **관찰 가능한 학급 전체 modulation 벡터**를 만들어야 하며, 그 변화량은 **실증 연구**에 근거해야 함. 임의 숫자가 아니라:
1. **방향**: 문헌이 명시적으로 말함 ("불안 ↑")
2. **대략적 범위**: Effect size (Cohen's d) 또는 퍼센트 변화
3. **대상 파라미터**: 어느 감정/인지 파라미터에 작용하는지

이후 autoresearch가 **범위 안에서** 정확한 값을 찾음.

---

### 25.10.2 시험 스트레스 (Exam Stress)

#### 이론적 근거
- **Yerkes-Dodson Law** (Yerkes & Dodson, 1908): 각성-수행 역U자 관계. 적당한 스트레스는 수행을 올리지만 과도하면 떨어뜨림.
- **Test Anxiety Theory** (Spielberger, 1980): Worry + Emotionality 이중 구조.
- **Working Memory under Stress** (Beilock & Carr, 2005): 스트레스가 작업기억을 점유 → 복잡한 과제 성능 저하.

#### 핵심 논문
- **Cassady & Johnson (2002)** "Cognitive test anxiety and academic performance." Contemp Educ Psychol 27(2):270-295
  - Test anxiety가 학업 성취에 미치는 효과 **d ≈ -0.40 ~ -0.52**
- **Hembree (1988)** "Correlates, causes, effects, and treatment of test anxiety." Rev Educ Res 58(1):47-77
  - 메타분석: test anxiety ↔ 학업 성취 **r ≈ -0.23**
- **Putwain (2008)** "Deconstructing test anxiety." Emot Behav Difficulties 13(2):141-155
  - ADHD 아동은 일반 아동보다 test anxiety가 **1.5-2배** 높음

#### 파라미터 변화 (modulation)
```yaml
exam_stress:
  global_modulation:
    anxiety: +0.15 ~ +0.30      # 학급 전체 기본 불안 증가
    frustration: +0.05 ~ +0.15  # 수행 압박
  cognitive_modulation:
    att_bandwidth: -1            # 작업기억 점유 (Beilock)
    importance_trigger: -30      # 반추 증가
  observable_modulation:
    distress_level: +0.10 ~ +0.20
    attention: -0.05 ~ -0.15    # 역U자: 경미한 경우 오히려 +
  adhd_amplification: 1.5 ~ 2.0  # ADHD는 1.5-2배 강하게 반응
```

#### 방향성 (논문 지원)
- 일반 학생: attention 소폭 감소, anxiety 증가
- **ADHD 학생**: attention 급격히 감소 + escalation ↑ (ADHD-H의 경우)
- 불안 프로파일: 증상 증폭 (기본 anxiety에 +0.2 더하면 dysfunctional level)

---

### 25.10.3 학기 주기 피로 (Academic Fatigue / Time-on-Task)

#### 이론적 근거
- **Mental Fatigue Theory** (van der Linden et al., 2003): 지속적 인지 활동은 전전두엽 자원을 고갈시킴
- **Vigilance Decrement** (Warm et al., 2008): 지속 주의 과제에서 시간이 갈수록 정확도 감소

#### 핵심 논문
- **Lim & Kwok (2016)** "The effects of varying break length on attention and time on task." Hum Factors 58(3):472-481
  - 지속 작업 60분 후 주의력 **d = -0.38**
- **Ackerman (2011)** "Cognitive fatigue: multidisciplinary perspectives on current research and future applications."
  - 학기 말 학업 fatigue → 과제 수행 전반 저하

#### 파라미터 변화
```yaml
academic_fatigue:  # turn 650+ 에서 점진적 증가
  global_modulation:
    excitement: -0.05 ~ -0.15
    loneliness: +0.02 ~ +0.08
  cognitive_modulation:
    att_bandwidth: -1 (after turn 800)
    impulse_override: +0.03 ~ +0.08
    task_initiation_delay: +0.05 ~ +0.15
  observable_modulation:
    attention: -0.05 ~ -0.12
    compliance: -0.03 ~ -0.08
```

---

### 25.10.4 또래 갈등 Event (Peer Conflict)

#### 이론적 근거
- **Social Pain Theory** (Eisenberger et al., 2003): 사회적 거부는 물리적 고통과 **동일한 뇌 영역** 활성화 (dACC, anterior insula)
- **Social Monitoring System** (Pickett et al., 2004): 위협 감지 후 과잉경계 모드

#### 핵심 논문
- **Eisenberger, Lieberman, & Williams (2003)** "Does rejection hurt? An fMRI study of social exclusion." Science 302(5643):290-292

### 25.10.7 가정 Stressor (Home Background)

#### 이론적 근거
- **ACE Study** (Felitti et al., 1998): Adverse childhood experiences → 장기 stress response system 영향

- **Blair & Raver (2012)** "Child development in the context of adversity." Am Psychol 67(4):309-318
  - Adversity → 실행기능 발달 지연

#### 구현 방법 — Hidden Stressor State
```python

# 일부 학생(5-10%)에게 높은 값
```

#### 파라미터 변화 (모든 턴, 지속 효과)

> "We include a 'home stressor' hidden profile as a distractor to test the specificity of ADHD identification: whether teachers (rule-based vs. LLM) correctly differentiate environmental stress from neurodevelopmental ADHD symptoms."

### 25.10.11 통합 구현 — Modulation Stack

여러 상황이 **동시에** 발생할 수 있음. 구현 전략:

```python
class SituationalModulator:
    def compute_modulation(self, turn: int, env_state: dict) -> dict:
        """Combine all active modulations additively."""
        mod = defaultdict(lambda: defaultdict(float))
        
        # 1. Long-term: 학기 주기
        mod_academic = self._academic_cycle(turn)
        
        # 2. Daily: 시간대
        mod_diurnal = self._diurnal_rhythm(env_state["period"])
        
        # 3. Event-based: 이벤트 스케줄
        mod_events = self._active_events(turn)
        
        # 4. Per-student: home stressor
        mod_home = self._home_stressor()  # dict of student_id -> mod
        
        # Combine (additive with clamping)
        return combine_modulations([mod_academic, mod_diurnal, mod_events], mod_home)
```

**중요**: 각 modulation은 **독립적으로 개발/테스트 가능**해야 함 → ablation 실험에 활용.

### 25.10.12 Autoresearch Calibration 연계

**범위만 주고 값은 autoresearch가 결정**:

```yaml
# .harness/situational_ranges.yaml
modulations:
  exam_stress:
    global.anxiety: [0.10, 0.35]       # 범위
    adhd_amplification: [1.2, 2.5]
    observable.attention: [-0.20, -0.02]
  
  peer_conflict:
    target.anger: [0.15, 0.45]
    target.escalation_risk: [0.10, 0.35]
    duration_turns: [20, 120]  # 지속 시간도 탐색
```

Autoresearch 루프:
1. 범위 내 값 샘플
2. 950턴 시뮬 실행 (다양한 situational events 포함)
3. Calibration target과 비교 — **상황별 정확도** 포함
4. 수렴할 때까지 반복

**논문 기여 재강화**:
> "Our calibration jointly optimizes baseline parameters AND situational modulation coefficients, ensuring the model reproduces both aggregate epidemiological targets AND situation-specific accuracy patterns (e.g., decreased teacher identification accuracy under exam stress)."

## 결론

**논문 방어 문장**:
> "Situational modulations were calibrated using evidence-based effect sizes from the literature. Specifically, we set prior ranges for each modulation based on meta-analytic findings (e.g., Hembree 1988 for test anxiety; Eisenberger et al. 2003 for peer conflict neural correlates) and let the autoresearch loop identify exact coefficients within these literature-supported ranges. This approach ensures that both direction and approximate magnitude of situational effects are grounded in empirical research, while system-level calibration accounts for model-specific interaction effects."

→ 방어 강력. 리뷰어가 "어디서 왔는지" 물으면 문헌 + "범위 기반 탐색" 로 답 가능.

## 25.11 교사 감정 모델 — 근거 기반 초기값 (순환 회피)

### 25.11.1 설계 원칙

학생 모델과 달리, **교사의 감정 파라미터는 절대 autoresearch 로 튜닝하지 않는다**. 대신:

1. **초기값**: 실제 교사 연구 normative data에서 **고정**
2. **변화**: LLM 에이전트가 상황 맥락 기반으로 추론
3. **효과**: 순환 논증 회피 + 외부 literature ground + 교사 학습 기여도 분리

이 분리가 논문의 핵심 방어 포인트. Autoresearch가 "어떤 감정의 교사가 잘 하나" 를 탐색하면 순환이 발생하므로, 교사 쪽은 **외부 데이터로 pin** 한다.

### 25.11.2 교사 감정 차원 (7차원 제안)

현재 코드의 `TeacherEmotionalState` 는 4차원(patience/empathy/frustration/bias)이지만, 문헌 기반으로 **7차원으로 확장** 권장:

| 차원 | 의미 | 근거 척도 |
|---|---|---|
| `emotional_exhaustion` | 정서적 소진 | MBI-EE (Maslach & Jackson 1981) |
| `depersonalization` | 탈인격화 (학생 대상화) | MBI-DP |
| `personal_accomplishment` | 개인 성취감 | MBI-PA |
| `self_efficacy` | 자기 효능감 | TSES (Tschannen-Moran 2001) |
| `empathy` | 공감 능력 | Teacher Empathy Scale (Goroshit 2016) |
| `patience` | 인내심 | Schnitker 2012 patience scale |
| `job_stress` | 직무 스트레스 | TSI (Fimian 1984) / 한국 교사 척도 |

**왜 7차원인가**:
- 학생(8차원)과 유사한 granularity
- MBI 3차원이 교사 연구의 골드 스탠다드
- TSES, Empathy, Patience, Stress는 각기 다른 종단면 예측력

### 25.11.3 각 차원 초기값 — 구체적 문헌 근거

- **Maslach et al. (1996)** MBI Manual (3rd ed.) — normative data
- **이영만 (2012)** "초등학교 교사용 심리적 소진 척도 개발 및 타당화" 초등교육연구 25(1) — 한국 초등교사 M ≈ 2.1/5.0
- 정규화 (Likert 5점 → 0-1): 2.1/5 ≈ 0.42
- 논문 근거: 한국 초등교사의 평균적 정서 소진 수준

#### (2) depersonalization = **0.30**
- **Maslach et al. (1996)** MBI Manual
- **조환이, 김정환 (2016)** "초등교사의 심리적 소진과 직무열의의 관계" 한국교원교육연구 33(2) — M ≈ 1.5/5.0

#### (3) personal_accomplishment = **0.72**

- 한국 초등교사는 일반적으로 직업 성취감이 상대적으로 높음 (전문직 인식)

```python
}
```

### 25.11.5 학생과의 핵심 차이

| 항목 | 학생 | 교사 |
|---|---|---|
| 초기값 형태 | **범위** (autoresearch 탐색 공간) | **고정값** (문헌 기반) |
| Autoresearch 대상 | ✅ 예 | ❌ 아니오 |
| 변화 로직 | 규칙 기반 (시뮬 내부) | **LLM 추론** (+ rule fallback) |
| 차원 수 | 8 | 7 |
| 역할 | 시뮬 대상 | 학습 주체 |
| 목적 | 현실 재현 | 학습 능력 관찰 |

**왜 이 차이인가**:
- 학생은 "현실의 다양한 아이들"을 재현해야 하므로 → 분포가 필요 → 범위 + calibration
- 교사는 "한 명의 에이전트가 학습" 하는 주체이므로 → 단일 고정 출발점 + 학습 궤적

### 25.11.6 변화 로직 — LLM-driven + Hybrid Fallback

**현재 (rule-based, 하드코딩)**:
```python
def update_after_turn(self, classroom_mood, n_incidents):
    self.patience -= 0.01 * n_incidents  # 하드코딩
    if classroom_mood == "chaotic":
        self.patience -= 0.005
```

**목표 (LLM-driven + fallback)**:
```python
class HybridTeacherEmotion:
    def update(self, context):
        """
        Routine turns: rule-based (빠름, 토큰 절약)
        Major events: LLM reasoning (맥락 판단)
        """
        if context.has_major_event or context.turn % 50 == 0:
            delta = self._llm_infer_update(context)
        else:
            delta = self._rule_based_update(context)
        self._apply_delta(delta, clamp=True)
```

**주요 이벤트 (LLM 호출)**:
- ADHD 식별 성공/실패
- 학생 위기 상황
- 학급 mood 급변
- 공존질환 학생 첫 대면
- 학기 주요 시점 (시험 전, 학기 말)

**일상 업데이트 (Rule-based)**:
- 일반 수업 턴
- 경미한 incident
- Baseline drift (시간에 따른 자연 변화)

### 25.11.7 로컬 모델 제약 대응

**로컬 LLM 사용 확정** (외부 API 아님):

**프롬프트 설계**:
```
SYSTEM: You are modeling the emotional state of a Korean elementary 
teacher. Given the current state and recent events, output JSON.

USER: Current state: {emotional_exhaustion: 0.45, ...}
Recent events: [Student S07 had major meltdown, class mood tense]
Output JSON with emotion deltas in [-0.1, 0.1].

ASSISTANT: {"emotional_exhaustion": +0.03, "patience": -0.02, ...}
```

**효율성 기법**:
1. **Delta 범위 강제** (-0.1 ~ +0.1 per turn) — LLM이 극단으로 가는 걸 방지
2. **JSON schema 강제** — 파싱 실패 방지
3. **Temperature 낮음** (0.2) — 재현성
4. **실패 시 rule-based fallback** — 시뮬 멈추지 않음
5. **배치 호출** — 여러 차원을 한 번에 업데이트

**비용/시간 예산**:
- 950턴 × LLM 호출 (매 턴) = 너무 비쌈
- 950턴 × LLM 호출 (major event만, ~50회) = 허용 가능
- 나머지 900턴은 rule-based drift

### 25.11.8 논문에 쓸 수 있는 문장

> "Teacher emotional parameters were not subject to autoresearch calibration. Initial values were fixed from validated teacher psychology literature (Maslach Burnout Inventory; Teacher Sense of Efficacy Scale; Korean teacher normative data). Dynamic emotional updates during simulation were delegated to the teacher's LLM agent, which interprets situational context (e.g., student incidents, classroom mood shifts, identification outcomes) to determine emotional shifts. For computational efficiency, routine turn updates used rule-based drift while major events triggered LLM reasoning. This explicit separation between simulator calibration (autoresearch-driven) and teacher emotional state (literature-fixed + LLM-dynamic) ensures that our claims about experience-based learning are not confounded with tuned emotional responsiveness."
